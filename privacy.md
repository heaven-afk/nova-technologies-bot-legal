# Privacy Policy

Nova Technologies values privacy and transparency.

## Data Collection
Our bots may collect:
- Server IDs
- Channel IDs
- Role IDs
- User IDs (only when required for functionality)

We do not collect:
- Personal messages
- Emails
- Passwords
- IP addresses

## Data Usage
Data is used strictly for:
- Bot configuration
- Command execution
- Moderation features

## Data Sharing
We do not sell, share, or distribute data to third parties.

## Data Removal
Server administrators may request data removal through the official support server.

## Updates
This policy may be updated as functionality evolves.
